# Logo Design Brief — Nomad Lens

## About the Product

**Nomad Lens** is a web-based tool for digital nomads, remote workers, and location-independent professionals. It helps users compare and rank countries based on quality-of-life factors — Economy, Healthcare, Education, Environment, Climate, Safety, Internet & Infrastructure, Happiness, and Human Development. Users adjust weighted sliders to re-rank countries based on what matters most to them personally. The app also features an interactive world map, budget matching, nomad visa comparison, and detailed country profiles.

The core idea: **look at any country through your own personal "lens" of priorities** — hence the name _Nomad Lens_.

## Target Audience

- Digital nomads choosing their next destination
- Remote workers comparing relocation options
- Expats researching quality of life across countries

## Brand Personality

- **Analytical** — data-driven, smart, trustworthy
- **Global** — worldwide scope, borderless mindset
- **Empowering** — the user is in control of their own criteria

## Visual Context

The current site uses a dark theme (`#0F1114` / `#0D0D0F` backgrounds), with clean typography, card-based layouts, and a muted color palette with accent highlights. The UI is built with Tailwind CSS and uses Lucide icons (thin, geometric line icons).

## Logo Requirements

1. **Style:** Minimalistic and modern. Clean geometry, no unnecessary detail.
2. **Concept direction (suggestions, not mandatory):**
   - A **lens / magnifying glass** combined with a **globe or map pin** — representing exploration through a personal filter.
   - Alternatively, abstract geographic shapes (parallels, meridians) viewed through a circular frame.
   - The letter **"N"** or initials **"NL"** could be subtly integrated if going with a logomark.
3. **Formats needed:**
   - Logomark (icon only) — for favicon, mobile, app icon contexts.
   - Logotype (icon + "Nomad Lens" wordmark) — for the site header and marketing.
4. **Color:**
   - Primary version on dark backgrounds (`#0F1114`).
   - Inverted / light-background version.
   - Monochrome (single-color) version.
5. **File deliverables:** SVG (vector), PNG (transparent, @1x and @2x), and favicon `.ico`.
6. **Avoid:** Overly literal travel clichés (suitcases, airplanes, palm trees). Keep it abstract and professional.
